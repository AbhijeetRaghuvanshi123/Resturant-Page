# Restaurant Page Project

This project is a restaurant page that showcases our menu, specials, and ambiance. The primary technologies used are:

- **JavaScript** (67.1%): The dynamic elements and interactions of the page are powered by JavaScript, enhancing user experience.
- **CSS** (26.9%): The styling and layout are created using CSS to provide an appealing look and feel.
- **HTML** (6%): The structure of the page is built with HTML, serving as the foundation for the content.
